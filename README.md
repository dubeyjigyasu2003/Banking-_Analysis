# Banking Analytics Dashboard

## Project Overview

The Banking Analytics Dashboard is an Excel-based data analytics project designed to monitor and evaluate loan portfolio performance. The dashboard provides a comprehensive view of lending activities, customer demographics, portfolio risk, repayment behavior, and key business metrics through interactive visualizations and KPI tracking.

This project demonstrates practical skills in data analysis, dashboard development, business intelligence, and data visualization using Microsoft Excel.

## Key Business Insights

### Portfolio Performance

The loan portfolio exceeded ₹681 Million in total funded amount, demonstrating significant lending activity across branches and products.

### Delinquency Risk

The delinquency rate stands at approximately 10.83%, highlighting the need for continuous repayment monitoring and collection management.

### Customer Demographics

Borrowers aged 26–35 represent the largest customer segment, making them a key target group for lending products.

### Credit Quality

Loan grades provide valuable insights into borrower creditworthiness and portfolio quality.

### Verification Risk

A considerable number of non-verified loans indicate potential areas of operational and credit risk.
